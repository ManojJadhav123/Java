# Introduction to Java

![Java Logo](../assets/java-logo.jpeg)

## What is Java?
Java is a **technology** that consists of both:
- A **programming language**
- A **software platform**

To create applications using Java, you need the **JDK (Java Development Kit)**, which is available for:
- Windows
- macOS
- Linux

---

## Languages that Run on the Java Platform
The Java platform supports many other programming languages, including:

1. Clojure
2. Groovy
3. JRuby
4. Jython
5. Kotlin
6. Scala

…and **50+ other languages**.

---

## Where Java is Used
Java can be used to develop a wide variety of applications, such as:
- Laptop and desktop applications
- Data center and enterprise applications
- Game consoles
- Scientific supercomputers
- Mobile phones
- Backend and server-side systems

---

## Popularity of Java
- Java is the **world’s third most popular programming language**
- Ranked after **Python** and **C**
- According to the **TIOBE Index (Jan 2026)**

---

## Java as a Programming Language

- Java is a **high-level**, **class-based**, and **object-oriented** programming language
- It is a **general-purpose** programming language
- Compiled Java code can run on **all platforms** (platform independent)
- Java was developed using **C++**
- Java syntax is similar to **C and C++**, but it provides **fewer low-level features**, making it safer and easier to use

---

## Key Takeaways
- Java = Language + Platform
- JDK is required for development
- Platform independent (Write Once, Run Anywhere)
- Widely used across industries
